# Canva Template Specs (build once per brand, then fill-in-the-blank)

A small fixed template set per brand so daily social is fast and on-brand. Build these in a Canva Brand Kit per brand (logos, ⚑ colors, fonts). Sizes are current platform specs.

## Per-brand brand kit (set up first)

- Logo (light + dark), ⚑ color palette (primary/secondary/neutral), 2 fonts (display + body), photo-treatment rule (warm, dignified, consented).

## Template set (build for each brand)

| Template | Size (px) | Use | Elements |
|---|---|---|---|
| **Reel/Short cover** | 1080×1920 | IG Reels, TikTok, YT Shorts | bold hook text top, brand logo, room for video |
| **Carousel** | 1080×1350 (4:5) | IG/FB/LinkedIn carousels | slide 1 hook, 2–6 body, last = CTA; consistent footer |
| **Quote card** | 1080×1080 | quotes, stats, milestones | big quote, attribution, logo corner |
| **Donor spotlight** | 1080×1080 | gratitude pillar | "thank you {name}", impact line |
| **Story** | 1080×1920 | IG/FB stories | CTA sticker zone, swipe-up/link |
| **Pinterest pin** | 1000×1500 (2:3) | Pinterest | tall, text overlay, link |
| **Impact stat** | 1080×1080 | education pillar | one big number + context |

## Rules

- **One idea per graphic.** Hook in the first 3 words.
- **Readable on mobile** at thumbnail size (big type, high contrast).
- **Consistent footer/logo** placement = instant brand recognition.
- **Consented photos only** (see `data/beneficiary-intake.md`); alt text on every post.
- Keep a **"swap zones"** convention: text + image are the only things that change per post; layout stays fixed → fast production.

## Workflow tie-in

In the weekly cycle (SOP-1), Wed = "design from templates": duplicate the right template, drop in the caption hook + consented asset, export, schedule. No design-from-scratch.
